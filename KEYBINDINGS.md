# Keybindings

- `Space` ‚Äî Play/pause transport.  
- `Shift+Space` ‚Äî Jump to loop start; resume if it was playing.  
- `Enter` ‚Äî Seek to start (temporarily disables loop for the jump).  
- `C` ‚Äî Toggle loop; auto-fills loop end if empty.  
- `B` ‚Äî Bounce: if loop is on, bounce loop range; otherwise bounce whole project (start to last clip).  
- `S` ‚Äî Save session to `config/last_session.json`.  
- `L` ‚Äî Load selected library item into the session (fallback to `config/test.wav` if none).  
- `Delete`/`Backspace` ‚Äî Delete selected clip(s) (when not renaming).  
- `F7` / `F8` / `F9` ‚Äî Toggle engine / cache / timing logs.

Library rename:
- Double-click a library item to enter rename mode.  
- Arrow keys move cursor; type to insert; `Backspace`/`Delete` to remove chars.  
- `Enter` commits rename on disk; `Esc` or clicking elsewhere cancels.
